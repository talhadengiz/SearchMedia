# Hepsiburada Mobile Team Case Study 

## Project

This project is a simple movie, music, app and book search app. You can reach the details of what you want from the results.

## Technologies

- MVVM
- Jetpack Navigation
- Safe Args
- Coroutines
- ViewModel
- LiveData
- Retrofit
- Lottie
- Paginate
